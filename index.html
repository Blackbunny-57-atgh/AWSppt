Architecture

Student → API Gateway → Lambda DynamoDB → SES → Teacher

The flow of data from student registration to teacher notification through AWS services: S3 hosts the form, API Gateway routes requests, Lambda processes data, DynamoDB stores records, and SES sends notifications.
